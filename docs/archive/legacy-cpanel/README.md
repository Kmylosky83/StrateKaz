# Documentacion Legacy cPanel (DEPRECADA)

> **AVISO:** Esta documentacion esta deprecada desde v3.7.2 (22 Enero 2026).
> El sistema ahora usa VPS con Nginx + Gunicorn.

## Razon de la Depreciacion

El proyecto migro de hosting compartido cPanel a VPS Hostinger por:

1. **Mayor control**: Configuracion completa del servidor
2. **Mejor rendimiento**: Gunicorn + Nginx optimizados
3. **Escalabilidad**: Facil agregar nuevas instancias
4. **Costos**: Mejor relacion costo/beneficio para multi-tenant
